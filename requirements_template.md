## 1. 프로젝트 개요

### 1.1 프로젝트 목표
> 이 앱은 기존 인바디 어플을 사용 할때 기록과 측정에 목적을 두는 반면 이 어플은 기록 뿐만 아니라 앱과 상화작용하여 성장을 가능하는 기능을 추가하는 앱입니다.


### 1.2 대상 사용자
> 학생, 운동선수, 운동 처음하는 사람 등


### 1.3 핵심 가치
> 기존 인바디 어플은 생성형 AI와 연동 되지 않아 궁금한 점이 있다면 AI 어플이나 전문가에게 질문 해야 하는 번거러움이 있지만
기존 앱과 다르게 상세한 기록과 그 기록을 바탕으로 AI와 상화작용하여 도움을 받습니다.

## 2. 기능 요구사항 (Functional Requirements)

> 시스템이 *무엇을 하는가*를 정의한다. 최소 5개 이상 작성
> 문장 패턴: "사용자는 ~할 수 있다" / "시스템은 ~한다"

### FR-01: [인바디 데이터 입력 및 시각화]

- **설명**: 사용자는 체중, 골격근량, 체지방률 데이터를 앱에 기록할 수 있고, 시스템은 이를 저장하여 변화 그래프를 보여준다.
- **수용 조건**:
  - [사용자는 소수점 첫째 자리까지 데이터를 직접 입력하거나 블루투스로 연동할 수 있어야 한다.]
  - [시스템은 저장된 데이터를 바탕으로 주간/월간/연간 단위의 라인 그래프를 끊김 없이 화면에 출력해야 한다.]

### FR-02: [제미나이 AI 맞춤형 종합 분석]

- **설명**: 사용자는 현재 신체 상태에 대한 분석을 요청할 수 있고, 시스템은 제미나이를 통해 맞춤형 피드백 리포트를 생성한다.
- **수용 조건**:
  - [사용자는 리포트 생성 버튼을 눌러 즉시 분석을 시작할 수 있어야 한다.] 
  - [시스템은 과거 데이터와 현재 데이터를 비교하여 골격근량 증가나 체지방 감소 등의 변화를 요약한 텍스트 리포트를 5초 이내에 화면에 표시해야 한다.] 

### FR-03: [제미나이 연동 실시간 건강 상담 챗봇]

- **설명**: 사용자는 식단 및 운동에 대해 AI와 실시간으로 대화할 수 있고, 시스템은 인바디 데이터를 기반으로 답변한다.
- **수용 조건**:
  - [사용자는 채팅창을 통해 자연어로 질문(예: "내 근육량에 맞는 단백질 섭취량은?")을 전송할 수 있어야 한다.] 
  - [시스템은 제미나이 API를 호출할 때 사용자의 최신 인바디 프로필을 함께 전달하여 개인화된 답변을 생성해야 한다.] 

### FR-04: [AI 추천 건강 목표 설정]

- **설명**: 사용자는 달성하고 싶은 다이어트나 벌크업 목표를 설정할 수 있고, 시스템은 제미나이의 조언을 곁들인 진행률을 관리한다.
- **수용 조건**:
  - [사용자는 목표 체중 및 목표 기간을 설정할 수 있어야 한다.] 
  - [시스템은 목표치가 건강상 위험한 수준(예: 한 달 만에 20kg 감량 등)일 경우, 제미나이의 가이드를 통해 경고 메시지와 대안 수치를 화면에 제안해야 한다.] 

### FR-05: [제미나이 가이드 기반 일일 투두 리스트(To-Do List)]

- **설명**: 사용자는 AI가 추천한 오늘의 운동과 식단을 할 일 목록에 등록할 수 있고, 시스템은 수행 여부를 체크한다.
- **수용 조건**:
  - [사용자는 제미나이가 추천한 일일 루틴 중 원하는 항목을 선택하여 나의 투두 리스트로 내보낼 수 있어야 한다.] 
  - [시스템은 사용자가 오늘 등록된 모든 투두 리스트를 체크(완료)하면, 제미나이가 생성한 칭찬 메시지와 격려 팝업창을 화면에 띄워야 한다.]

## 3. 비기능 요구사항 (Non-Functional Requirements)

> *어떻게 동작하는가*를 정의한다. 직접 보이지 않지만 품질을 결정한다.

### 3.1 성능 (Performance)

- 메인 화면 진입 후 1초 이내에 첫 콘텐츠 및 인바디 요약 그래프 표시
- 제미나이 AI에게 질문 전송 후 3초 이내에 실시간 답변 출력 시작
- 인바디 데이터 수동 입력 또는 블루투스 연동 저장 완료 까지 0.5초 이내 처리

### 3.2 보안 (Security)

- 앱 실행 시 바이오 인증(Face ID/지문인식) 또는 비밀번호(PIN)를 통한 2차 잠금 기능 제공
- 제미나이 API 전송 시 사용자의 식별 정보(이름, 이메일 등)는 완전 비식별화(마스킹)하여 전송

### 3.3 사용성 (Usability)

- 스마트폰 시스템 설정의 폰트 크기 변경에 맞추어 UI 레이아웃이 깨지지 않고 자동 최적화 반영
- 처음 앱을 실행하는 사용자를 위해 3단계 이하의 직관적인 가이드 대화창(온보딩) 제공

### 3.4 호환성 (Compatibility)

- 안드로이드(Android 10 이상) 및 iOS(iOS 15 이상) 최신/구형 스마트폰 환경 동시 지원
